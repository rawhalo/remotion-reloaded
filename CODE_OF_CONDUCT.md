# Code of Conduct

## Our Commitment

We are committed to making participation in this project a respectful, harassment-free experience for everyone.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent and discuss ideas, not people.
- Give actionable technical feedback.
- Accept responsibility and correct mistakes.

## Unacceptable Behavior

- Harassment, discrimination, or abusive language.
- Personal attacks, trolling, or intimidation.
- Publishing private information without consent.
- Any conduct that is inappropriate in a professional setting.

## Scope

This Code of Conduct applies to project spaces, including issues, pull requests, discussions, and related community channels.

## Reporting

Report concerns to the maintainers privately through GitHub (direct message or private channel if available).

If immediate safety concerns exist, contact local authorities first.

## Enforcement

Maintainers may remove, edit, or reject comments, commits, code, issues, and other contributions that violate this Code of Conduct.

Repeated or severe violations may result in temporary or permanent bans from project participation.
