# Roadmap

This roadmap is a high-level view of planned capabilities beyond the currently shipped Phase 1 foundation.

## Current

### Phase 1: Technical Foundation

Status: **Shipped**

Focus:
- Deterministic animation workflows
- Effects and presets
- Three.js integration
- Project setup and diagnostics tooling

## Planned

### Phase 2: Cinematography and Motion

Status: **Planned**

Focus:
- Camera movement primitives
- Transition systems
- More procedural motion controls

### Phase 3: Intelligence Layer

Status: **Planned**

Focus:
- Style and mood abstractions
- Audio-reactive behavior
- Smarter layout and narrative tooling

### Phase 4: Production Scale

Status: **Planned**

Focus:
- Multi-variant output workflows
- Asset/data pipeline integrations
- Automated quality checks and analysis

## Notes

- This file is intentionally non-detailed and user-facing.
- Detailed technical planning remains in the phase spec documents.
- Scope and sequencing may change based on validation and user feedback.
